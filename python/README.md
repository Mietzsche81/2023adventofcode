# Prerequisites:
```
pip install poetry
```
# Install Dependencies:
```
cd 2023adventofcode/python
poetry install
```

# Run a given puzzle:
```
cd 2023adventofcode/python
poetry run python ./aoc2023/day[day] [part] [input_file]
```

Where:
- `day`: 1-25
- `part`: `1`, `2`, `A`, or `B`
- `input_file`: path to puzzle input
